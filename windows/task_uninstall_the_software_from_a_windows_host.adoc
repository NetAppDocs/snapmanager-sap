= Uninstall SnapManager
:icons: font
:imagesdir: ../media/

[.lead]
You can uninstall SnapManager from the host server.

== What you'll need

* Stop the host server before uninstalling SnapManager.

.Steps

. Click *Start* > *Control Panel* > *Add or Remove Programs*.
. Select *SnapManager for SAP*.
. Click *Uninstall*.
